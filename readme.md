Dit project heb ik bedacht, omdat een vriend van me een simpel project wou om in python te maken. Hierop heb ik geantwoord dat je een simpele rekenmachine kan maken met 3 moeilijkheidsgraden erin. de eerste is dat je 2x input vraagt voor 2 getallen en deze bij elkaar optelt en het antwoord toont, de tweede is hetzelfde maar je vraaft of de user plus of min wil en de 3e is hetzelfde maar je vraagt of de user +, -, * of / wil doen.
